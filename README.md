# BeMoRe
A humanoid bot capable of moving like a human being, using articulations to create the movement


# Needed

1. 20 Servos (power between 9g and 20 Kg)
2. 70 mf jumper cables
3. 1 esp-32 (or Arduino Giga R1) --> will use esp-32 (cheaper)
4. 15 metal plates, with holes for screws
5. 2 curved metal plates
6. 1 h-bridge
7. 1 battery-pack

# Time schedule

1. 2 weeks for completing the legs
2. 2-3 weeks for completing arms
3. 1 week to make it simulate his shutdown
